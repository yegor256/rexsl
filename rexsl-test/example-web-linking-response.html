Web Linking Response

RFC 5988 “Web Linking” recommends to use Link HTTP header for linking pages. WebLinkingResponse implements parsing of such header. This is how you can use it:

URI destination = new JdkRequest("http://my.example.com")
  .fetch()
  .as(WebLinkingResponse.class)
  .link("next") // IOException here if such a link doesn't exist
  .uri();
