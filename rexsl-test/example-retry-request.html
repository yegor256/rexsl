HTTP Request that Retries on Failure

RetryRequest. is a decorator of Request that retries to fetch a response a few times before giving up and throwing IOException, for example:

Request request = new RetryRequest(new JdkRequest("http://www.google.com"));
String body = request.fetch().body();

If it recommended to use this decorator in production, to avoid accidental exceptions due to unstable connection.
